# Student-Placement-PowerBI-Dashboard

## Project Overview
This project presents a **one-page interactive Power BI dashboard** analyzing
college student placement data to identify key factors influencing placement outcomes.

## Dataset
- Academic Performance
- CGPA
- IQ
- Communication Skills
- Internship Experience
- Projects Completed
- Placement Status

## Dashboard Features
- Total Students & Placement Percentage
- Placement Distribution (Placed vs Not Placed)
- Internship Impact on Placement
- CGPA & Academic Performance Analysis
- Communication Skills & Projects Influence
- Interactive slicers for filtering data

## Tools Used
- Power BI
- Microsoft Excel
- GitHub

  1. Download the `.pbix` file
  2. Open it using Power BI Desktop
  3. Interact with filters and visuals
